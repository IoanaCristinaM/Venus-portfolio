The Bug Button

The MindBugs team has proposed a collaboration to create a visual design for a button to be used in their application. The button must be clearly visible and understandable to users, indicating clear forward and backward movements in the game, as well as left and right. The design of this button must also respect the theme of the game, so it should present elements of the Steampunk imaginary.

On the user interface (UI) side, the button is designed to resemble a light bulb. Thanks to the Steampunk elements attached to it, such as copper wires and bolts, it takes on the shape and resemblance of an insect or bug (hence the name MindBugs).
To indicate the use of these assets, the button incorporates an arrow into its design to aid in understanding its interaction and role in the user experience. I incorporated the arrow model because it is a symbol familiar to users.

Image Description

The button is positioned at the bottom of the application interface screen, as can be seen in the image, under the presentation of a game tutorial. Here, users can interact with the button to move forward and backward to read the information. The buttons are mirrored to each other to indicate movement to the left or right.

Image Description

In the visual design of the button, I incorporated Steampunk elements by using a copper color palette, as well as parts that resemble copper wires or bolts to create the shape of an insect. The button is built around a light bulb, inspired by fireflies, so when users interact with the button, it lights up or shows a lighting movement to indicate it's being pressed.
